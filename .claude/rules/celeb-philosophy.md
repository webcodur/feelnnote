# 셀럽 감상 철학 생성 룰북

인물의 콘텐츠 감상 철학을 작성하는 가이드.

---

## 작성 기준

**Persona: 심층 문화 심리학자**

단순한 취향 나열이 아닌, **"문화적 경험과 감상 패턴이 이 인물의 인생과 업적을 어떻게 구축했는가?"**를 분석해야 한다.

---

## 필수 포함 요소 (Fact-based)

### 1. 결정적 콘텐츠

인생의 방향을 바꾸거나(진로, 가치관), 큰 영감을 준 구체적인 작품명(책, 영화, 음악 등)을 **반드시 1개 이상** 언급한다.

### 2. 감상 방식의 특이점

단순히 "즐겼다"가 아니라 **"어떻게"** 즐겼는지 묘사한다.
- 예: 밑줄을 그으며 암기
- 예: 한 곡만 반복 청취
- 예: 성대모사

### 3. 직업/업적과의 연결고리

그가 즐긴 콘텐츠가 실제 그의 활동(정치, 예술, 사업 등)에 어떤 **구체적 영향**을 미쳤는지 인과관계를 서술한다.

---

## 금지 사항 (Strict Rules)

| 금지 | 설명 |
|------|------|
| ❌ **추측성 어미** | "~로 보인다", "~짐작할 수 있다", "~것 같다" 절대 금지. **모든 문장은 "~다."로 단정지어 끝낼 것.** |
| ❌ **추상적 나열** | "다양한 장르를 좋아했다" (X) → "러시아 대문호의 소설에 탐닉했다" (O) |
| ❌ **뻔한 칭찬** | 비판적 시각이나 모순된 취향이 있다면 그대로 서술하여 입체감 부여 |
| ❌ **'컨텐츠' 단어** | '컨텐츠', '콘텐츠' 단어 자체 사용 금지. 대신 구체적인 장르명(책, 영화, 음악, 작품 등)이나 '문화적 경험', '예술 향유' 등으로 표현 |

---

## 분량 규칙

- **최소 700자, 최대 900자**
- 자료가 부족한 인물도 **최소 3문단** 이상 작성
- 자료 부족 시 해당 인물의 직업적 특성, 시대적 맥락, 유추 가능한 감상 태도로 보완

## 모범 답안

**아래 예시의 문체, 밀도, 호흡을 그대로 따른다.**

```


알렉산더 대왕은 책을 지식의 저장소나 교훈집으로 인식하지 않는다. 그는 텍스트를 자기 정체성을 규정하는 거울로 사용한다. 독서는 이해의 과정이 아니라 자기 동일시의 과정이다. 이야기 속 인물과 자신을 분리하지 않고, 그 자리에 스스로를 대입한다. 이 태도는 감상을 현실과 단절된 행위로 두지 않는다.

그가 반복적으로 붙들고 있던 호메로스의 일리아스는 영웅이 어떻게 기억되는가를 다룬 서사이다. 알렉산더는 이를 관찰자의 시선으로 읽지 않는다. 아킬레우스를 분석하지 않고, 아킬레우스가 되기로 선택한다. 이 선택은 사유에 머물지 않고 행동으로 즉각 전환된다. 마케도니아 군의 쐐기대형은 왕이 후방에서 명령하는 구조가 아니라, 왕이 최전선에서 돌파를 주도하는 전술이다. 그는 지휘관이기 이전에 서사의 주인공으로 전장에 선다.

알렉산더가 직접 최전선에서 돌격하고 중상을 입는 행위는 무모함이 아니라 감상의 귀결이다. 영웅 서사에서 주인공은 안전한 위치에 머물지 않는다. 결정적 장면에는 반드시 몸을 던진다. 그는 이 문법을 현실 전쟁에 그대로 적용한다. 책에서 읽은 영웅의 자리를 전장에서 재현함으로써, 텍스트와 현실의 경계를 제거한다.

이러한 감상 태도는 그의 삶 전체를 관통한다. 그는 제국의 지속 가능성보다 이야기의 완결성을 우선한다. 책은 그에게 참고 자료가 아니라, 따라야 할 형식이다. 알렉산더 대왕은 위대한 독서가가 아니라, 읽은 내용을 그대로 살아낸 인물이다. 그의 감상은 해석이 아니라 실천이다.
```

## 자료 수집

### 1단계: DB 기존 콘텐츠 우선 활용

작성 전 해당 셀럽의 수집된 콘텐츠를 먼저 조회한다.

```sql
SELECT c.title, c.creator, c.type, uc.review
FROM user_contents uc
JOIN contents c ON c.id = uc.content_id
WHERE uc.user_id = '{셀럽ID}'
ORDER BY c.type;
```

- **review 필드**: 셀럽이 해당 작품을 언급한 맥락·발언이 담겨 있다. 철학 작성의 핵심 소스로 활용한다.
- **작품 목록**: 장르별 분포, 반복 등장 주제 등을 파악하여 감상 패턴 분석에 활용한다.

### 2단계: 웹 검색 (보충)

| DB 콘텐츠 | 검색 전략 |
|-----------|----------|
| **10건 이상** | 검색 1~2회. 감상 태도·에피소드·명언 중심으로 보충 |
| **10건 미만 또는 없음** | 최소 2~3개 검색어로 다각도 조사 |

### 검색 키워드 예시

- "{셀럽명} 독서 책 추천 인터뷰"
- "{셀럽명} 영화 음악 취미"
- "{셀럽명} 명언 철학"
