# 셀럽 프로필 생성 룰북

Feelandnote 플랫폼의 셀럽 프로필 생성 가이드. Claude Code agent와 web-bo에서 공통 참조.

---

## 1. 기본 정보 (Basic Profile)

### JSON 출력 형식

```json
{
  "nickname": "셀럽 이름 (한국어로 작성, 예: 슬라보예 지젝)",
  "profession": "직군 코드",
  "title": "수식어 (2~8자, 대표작명/호칭/업적 중 하나)",
  "nationality": "국가 코드 (ISO 3166-1 alpha-2, 예: US, KR, GB, JP)",
  "gender": "성별 (남성: true, 여성: false, 불명: null)",
  "birth_date": "출생일 (YYYY-MM-DD 또는 -356 같은 기원전 연도)",
  "death_date": "사망일 (생존시 빈 문자열)",
  "bio": "인물 소개글 (2줄 분량, 한국어)",
  "quotes": "대표 명언 (한 문장)",
  "avatar_url": "썸네일 이미지 URL (선택사항)",
  "portrait_url": "초상화 이미지 URL (선택사항)",
  "is_verified": false
}
```

### 직군 코드

- `leader` (지도자)
- `politician` (정치인)
- `commander` (지휘관)
- `entrepreneur` (기업가)
- `investor` (투자자)
- `humanities_scholar` (인문학자)
- `social_scientist` (사회과학자)
- `scientist` (과학자)
- `artist` (예술인)
- `author` (작가)
- `actor` (배우)
- `influencer` (인플루엔서)
- `athlete` (스포츠인)
- `other` (기타)

### 수식어(title) 작성 가이드

- **용도**: 인물 이름 앞에 붙는 짧은 수식어 (예: "군주론의 마키아벨리", "철의 여인 마거릿 대처")
- **길이 제한**: 2~8자 이내로 최대한 짧게
- **우선순위** (위에서부터 적용):
  1. **대표작**: 작품명 하나 (예: '군주론', '자본론', '코스모스')
  2. **실제 호칭/별명**: 역사적 통용 호칭 (예: '철의 여인', '가왕', '독안룡')
  3. **핵심 업적**: 가장 상징적인 것 하나 (예: '테슬라 창립자', '지동설')
- **금지**: 긴 설명형 문장, 나열식 표현, "~의 아버지/어머니" 같은 뻔한 표현

### 성별(gender) 필드 규칙

- `gender` 필드는 **boolean** 타입
- **true**: 남성
- **false**: 여성
- **null**: 알 수 없음 또는 해당 없음 (논바이너리, 그룹, 불명)

성별 판단 기준:
- 역사적 인물: 문헌/기록 기반으로 명확히 판단
- 현대 인물: 공식 프로필 또는 본인 발언 기반
- 불명확한 경우: null로 설정

### 작성 규칙

1. `nickname`은 정확한 풀네임을 한국어로 작성 (예: 슬라보예 지젝, 알베르트 아인슈타인)
2. `bio`는 100자 이내. 주어 없이 출신/직업을 짧게 서술하고 마침표로 끊은 뒤 주요 업적을 이어감
3. `nationality`는 ISO 3166-1 alpha-2 국가 코드. 현대인이 아닌 경우 현재 해당 지역의 국가 코드로 재구성 (예: 알렉산더 대왕 → GR, 공자 → CN). 고대 국가로 특정 불가능한 경우 빈 문자열
4. `birth_date`/`death_date`는 정확한 날짜를 알 수 없으면 연도만 작성
5. `quotes`는 50자 이내, 알려진 것이 없으면 빈 문자열
6. 문자열 내 큰따옴표는 작은따옴표로 대체
7. **`is_verified`는 항상 false** (공식 인증은 수동 처리)

## 2. 영향력 평가 (Influence)

### JSON 출력 형식

```json
{
  "political": { "score": 0, "exp": "설명" },
  "strategic": { "score": 0, "exp": "설명" },
  "tech": { "score": 0, "exp": "설명" },
  "social": { "score": 0, "exp": "설명" },
  "economic": { "score": 0, "exp": "설명" },
  "cultural": { "score": 0, "exp": "설명" },
  "transhistoricity": { "score": 0, "exp": "설명" }
}
```

### 핵심 원칙: 인과적 기여도 평가

평가 기준은 **"해당 분야의 변화에 얼마나 인과적으로 기여했는가?"**

### 인과적 기여의 3단계

| 단계 | 정의 | 가중치 | 예시 |
|------|------|--------|------|
| **직접 기여** | 본인이 직접 창안·실행·구축 | 최고 | 아인슈타인의 상대성이론 |
| **구조적 기여** | 해당 분야 발전의 필수 조건·토대·프레임워크 제공 | 높음 | 무함마드 → 이슬람 문명권 → 중세 과학 |
| **촉매적 기여** | 해당 분야 발전을 촉진하는 환경·동기·자원 조성 | 중간 | 메디치 가문의 르네상스 후원 |

### 인과 연결 거리 규칙

- **1~2단계**: 직접 기여로 간주, 최고 점수 가능
- **3~4단계**: 구조적 기여로 인정, 중상위 점수
- **5단계**: 촉매적 기여로 인정, 중간 점수
- **6단계 이상**: 인정 불가 (과도한 간접 연결)

### 6개 영역 (각 0-10점)

#### 점수 등급

| 점수 | 기준 |
|------|------|
| **9-10** | 글로벌 스케일에서 직접 또는 구조적으로 패러다임 전환 |
| **7-8** | 글로벌/주요 권역에서 핵심 시스템 구축, 또는 필수적 구조적 토대 제공 |
| **5-6** | 국가/지역/분야 단위 실질적 영향, 또는 촉매적 기여로 주요 변화 유발 |
| **3-4** | 부분적 기여 또는 간접적 촉매 역할 |
| **1-2** | 미미한 영향 |
| **0** | 관련 없음 |

#### 영역별 평가 질문

1. **political** (정치·외교): 정치 체제/정책/외교 질서를 변화시켰는가?
2. **strategic** (전략·안보): 군사 전략/무기 체계/안보 패러다임을 변화시켰는가?
3. **tech** (기술·과학): 기술/과학 발전에 기여했는가?
4. **social** (사회·윤리): 사회 규범/윤리 담론/제도를 변화시켰는가?
5. **economic** (산업·경제): 산업 구조/경제 시스템을 변화시켰는가?
6. **cultural** (문화·예술): 문화/예술 트렌드/장르를 변화시켰는가?

#### 평가 예시

| 인물 | political | strategic | tech | social | economic | cultural |
|------|-----------|-----------|------|--------|----------|----------|
| 아인슈타인 | 3 | 9 | 10 | 4 | 8 | 2 |
| 무함마드 | 10 | 9 | 5 | 10 | 8 | 9 |
| 스티브 잡스 | 1 | 0 | 7 | 4 | 9 | 5 |
| 김정은 | 7 | 7 | 1 | 5 | 2 | 4 |
| BTS | 1 | 0 | 0 | 3 | 5 | 8 |
| 넬슨 만델라 | 9 | 2 | 0 | 9 | 4 | 5 |

### 통시성 (transhistoricity, 0-40점)

#### 정의

"과거부터 미래까지, 인류사 전체에 걸쳐 영향을 미치는가?"

#### 평가 기준

1. **역사적 영향**: 장기간에 걸쳐 인류에게 영향을 미쳤는가?
2. **미래 지속성**: 인류 문명의 구조적 토대로서 앞으로도 영향이 지속될 것인가?

#### 미래 영향 인정 조건

| 인정 | 불인정 |
|------|--------|
| 이미 자기강화적 시스템(종교, 법체계, 과학원리)으로 정착 | 아직 검증되지 않은 미래 가능성 추측 |
| 해당 시스템의 폐기가 문명적 단절을 의미할 정도로 통합됨 | "AI로 인류를 바꿀 것이다" 류의 예측 |

#### 점수 기준

| 점수 | 기준 | 예시 |
|------|------|------|
| **35-40** | 인류 문명의 근본 토대. 과거-현재-미래 전체에 영향 | 예수, 붓다, 공자, 무함마드, 아리스토텔레스 |
| **28-34** | 문명 패러다임 전환, 구조적 토대로 정착 | 뉴턴, 다윈, 아인슈타인 |
| **20-27** | 특정 분야의 근본 토대로 장기 지속 | 칸트, 마르크스, 프로이트 |
| **10-19** | 근현대에 큰 영향, 일부 구조적 기여 | 처칠, 간디, 케인스 |
| **5-9** | 현대에 영향력 있으나 장기 지속성 불확실 | 스티브 잡스, 일론 머스크 |
| **0-4** | 동시대에만 영향력 | 대부분의 현대 연예인·스포츠인 |

### 금지 사항

- ❌ 검증 불가능한 추측 ("~했을 것이다")
- ❌ 단순 상관관계 ("같은 시대에 ~가 발전했다")
- ❌ 6단계 이상 떨어진 간접 연결
- ❌ 희망적 미래 예측 ("앞으로 ~할 것이다")

### 출력 규칙

1. 각 영향력 exp는 **30자 이내 1문장**으로 간결하게
2. 알려진 정보만 반영, 추측 금지
3. 전문 분야 외 영역은 낮은 점수 부여
4. 문자열 내 큰따옴표는 작은따옴표로 대체

## 3. 감상 철학 (Consumption Philosophy)

### 작성 기준 (Persona: 심층 문화 심리학자)

단순한 취향 나열이 아닌, **"문화적 경험과 감상 패턴이 이 인물의 인생과 업적을 어떻게 구축했는가?"**를 분석해야 한다.

### 필수 포함 요소 (Fact-based)

1. **결정적 콘텐츠**: 인생의 방향을 바꾸거나(진로, 가치관), 큰 영감을 준 구체적인 작품명(책, 영화, 음악 등)을 반드시 1개 이상 언급
2. **감상 방식의 특이점**: 단순히 "즐겼다"가 아니라 "어떻게" 즐겼는지 (예: 밑줄을 그으며 암기, 한 곡만 반복 청취, 성대모사 등)
3. **직업/업적과의 연결고리**: 그가 즐긴 콘텐츠가 실제 그의 활동(정치, 예술, 사업 등)에 어떤 구체적 영향을 미쳤는지 인과관계 서술

### 금지 사항 (Strict Rules)

- ❌ **추측성 어미 금지**: "~로 보인다", "~짐작할 수 있다", "~것 같다" 절대 사용 금지. **모든 문장은 "~다."로 단정지어 끝낼 것.**
- ❌ **추상적 나열 금지**: "다양한 장르를 좋아했다" (X) → "러시아 대문호의 소설에 탐닉했다" (O)
- ❌ **뻔한 칭찬 금지**: 비판적 시각이나 모순된 취향이 있다면 그대로 서술하여 입체감을 부여
- ❌ **'컨텐츠' 단어 사용 금지**: '컨텐츠', '콘텐츠'라는 단어 자체를 사용하지 말 것. 대신 구체적인 장르명(책, 영화, 음악, 작품 등)이나 '문화적 경험', '예술 향유' 등으로 표현

### 구조 (4개 문단 구성)

1. **1문단 (정체성 & 결정적 계기)**: 인물의 핵심 정체성을 정의하고, 그것을 형성한 결정적인 콘텐츠 경험(책, 영화 등)을 서술
2. **2문단 (주력 분야 & 심층 태도)**: 그가 가장 깊이 파고든 콘텐츠 장르(문학, 역사, 기술 등)와 그것을 대하는 태도(학구적, 비판적, 쾌락적 등) 묘사
3. **3문단 (확장 & 의외성)**: 주력 분야 외의 취향(예: 정치인의 음악 취향, 과학자의 예술 취향)을 언급하며 인물의 입체성 부각
4. **4문단 (결론 & 인생관)**: 결국 그에게 '감상(Appreciation)'이란 무엇인지 한 문장으로 정의하고, 그것이 그의 삶을 어떻게 완성했는지 요약

### 자료 수집 전략

감상 철학 작성 전 DB에 이미 수집된 콘텐츠를 먼저 조회하여 활용한다. 상세 규칙은 `.claude/rules/celeb-philosophy.md`의 "자료 수집" 섹션 참조.

1. **DB 조회**: 해당 셀럽의 user_contents + contents에서 작품 목록과 review(맥락·발언) 확인
2. **웹 검색 보충**: DB 콘텐츠 10건 이상이면 1~2회, 미만이면 2~3회 검색

### 모범 답안 예시

```
블라디미르 푸틴의 감상 철학:

블라디미르 푸틴의 문화적 체험은 철저히 국가적 정체성과 강한 러시아라는 프레임 안에서 이루어진다. 그에게 미디어는 단순한 유희의 도구가 아니라 인생의 경로를 결정짓고 애국심을 고취하는 강력한 수단이다. 어린 시절 그를 정보기관의 길로 이끈 것은 1968년작 첩보 영화인 '방패와 칼'이었다. 이 영화는 그에게 조국 헌신의 로망을 심어주었으며, 작품이 단순한 허구가 아닌 현실의 의지를 단련하는 교본이 될 수 있음을 보여준 사례다.

문학과 역사를 대하는 태도 역시 러시아적 가치의 보존과 강화에 집중되어 있다. 그는 도스토옙스키와 톨스토이 같은 대문호들의 고전을 통해 러시아인의 영혼과 도덕적 우월성을 탐색한다. 특히 오마르 하이얌의 시집을 수시로 들여다보며 삶의 지혜를 구하는 것으로 알려져 있다. 역사적 사료에 대해서는 거의 집착에 가까운 진지함을 보이는데, 스스로 분석하고 논문을 작성할 정도로 학구적이다. 그에게 지식 습득은 권력을 정당화하고 국가의 방향을 설정하는 필수적인 기초 작업이다.

보수적이고 권위적인 성향은 예술 취향에서도 드러난다. 애국적인 가사를 담은 밴드 '루베(Lube)'의 음악을 즐겨 듣거나, 차이콥스키 같은 고전 음악을 향유하며 러시아 문화의 위상을 과시한다. 반면 게임 같은 최신 디지털 미디어에는 무관심하며, 대신 유도와 같은 무술을 정신 수양을 위한 정교한 수련으로 취급하여 평생 매진했다. 이는 그가 무질서한 재미보다는 규율과 질서가 담긴 형식을 선호함을 입증한다.
```
