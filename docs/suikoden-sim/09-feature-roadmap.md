# 09. 기능 로드맵

> 기획서(01~08) 대비 현재 구현 상태 분석 및 우선순위별 구현 계획.

---

## 현재 구현 상태 (2026-02-14)

### 완료

| 시스템 | 구현 내용 |
|--------|----------|
| **실시간 엔진** | 틱 시스템, 시간 진행, 계절, 배속(1x/2x/3x/정지) |
| **캐릭터 이동** | A* 경로탐색, 지형별 이동비용, 실시간 이동 |
| **건설** | 13종 건물 정의, 건설 명령 → 시공 → 완성, 맵 타일 반영, 철거 |
| **자원 생산** | 건물별 자원 생산, 근무자 1.5배 보너스, 기본 수입, 세율 보정 |
| **식량 소비** | 병력 기반 식량 소모, 부족 시 사기/충성 하락 |
| **근무 배치** | 캐릭터를 건물에 근무 배치 → 생산 보너스 |
| **순찰** | town 주변 랜덤 순찰 경로 |
| **인재 영입** | 방랑 인재 탐색, 성공률(주군 인애 + 명성 보너스), **명성 기반 영입 제한** |
| **자동 내정** | idle 캐릭터 자동 배치 (건설/근무/순찰) |
| **전투** | 턴제 전투(별도 BattleScreen), 병과 스킬, 지형 효과, 성벽 |
| **AI** | 세력별 성향(정복/책략/경제/인덕/문화), 실시간 AI 판단 |
| **세계맵** | 17개 영토, 7개 권역, 미니맵 표시, 영토 전환 |
| **영토 내부맵** | 16x12 타일맵, 지형 생성, 캐릭터 토큰, 건물 배치 |
| **UI** | GameHUD(상단), GameToolbar, CommandMenu(사이드), 컨텍스트 메뉴, 캐릭터 상세 |
| **아이템** | 콘텐츠 기반 장비 시스템 (타입/등급 정의) |
| **인사** | 훈련(연병장 필요), 포상(금 50→충성+10), 처벌(충성/사기 하락) |
| **외교** | 동맹(200금), 정전(100금), 조공(가변), 항복 권유(전력 30% 이하), 관계도 시스템 |
| **경영** | 세율 조정(낮음/보통/높음), 건물 철거, 민심 시스템, 인구 변동 |
| **이벤트** | 계절 이벤트 7종(봄~겨울), 랜덤 이벤트 4종, 월 1회 자동 발생 |
| **명성** | 0~1000 스케일, 등급별 영입 요구치, 건설/점령/영입/외교 트리거 |

## 구현 우선순위 (잔여)

### Phase A: UI 개선 + 이벤트 팝업

| 항목 | 설명 |
|------|------|
| **이벤트 팝업** | 이벤트 발생 시 화면 중앙 팝업, 확인 버튼 |
| **영토 비교** | 세력별 영토/자원/병력 비교 테이블 |
| **메뉴바 구조화** | GameHUD → 3줄 구조(메뉴/도구/상태) 분리 |

### Phase B: 인사/경영 확장

| 항목 | 설명 |
|------|------|
| **학당 학습** | 학당에 배치 → intellect/virtue 성장 |
| **징병** | 병영에서 인구→병력 전환, 민심 -5 |
| **민심 세분화** | 폭동(0-19) 시 건물 파괴/인재 이탈 이벤트 |
| **직위 시스템** | 태수, 군사 등 역할 → 해당 영토 보너스 |

### Phase C: 승리 조건 + 날씨

| 항목 | 설명 |
|------|------|
| **날씨** | 맑음/비/눈/폭풍 → 이동속도/전투력 보정 |
| **문화 승리** | 극장/사원 문화 영향력 → 전체 60% 이상 |
| **외교 승리** | 2/3 이상 세력과 동맹 체결 |
| **이간** | 금 300, 두 AI 세력 간 관계 악화 |

### Phase D: 아이템 활용

| 항목 | 설명 |
|------|------|
| **장비 장착 UI** | 캐릭터 상세에서 scroll/treasure 교체 |
| **아이템 획득** | 전투 승리/이벤트/교역으로 획득 |
| **전투 효과** | 장비 보너스 전투에 실제 반영 |
