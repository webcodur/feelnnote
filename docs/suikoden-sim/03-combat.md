# 03. 전투 시스템

## 전투 개요

**방식**: 턴제 택티컬 (수호지 천도 108성 참조)
**맵**: 헥스 또는 그리드 기반 전장 (12×8 ~ 16×12)
**턴 제한**: 공격 측 30턴, 방어 측 30턴
**참여 인원**: 세력당 최대 5명 출진

---

## 전장 구성

### 지형 타일

| 지형 | 이동 비용 | 방어 보너스 | 특수 |
|------|----------|-----------|------|
| 평지 | 1 | 0% | - |
| 숲 | 2 | +20% | 매복 가능 |
| 산 | 3 | +40% | 원거리 사거리 +1 |
| 강 | 2 | -10% | 도하 중 공격 시 피해 1.5배 |
| 성벽 | 진입불가 | +60% | 공성 장비 필요 |
| 성문 | 1 | +30% | 파괴 시 성벽 무력화 |
| 마을 | 1 | +10% | 점령 시 보급 회복 |

### 전장 유형

| 유형 | 구성 | 발생 조건 |
|------|------|----------|
| **야전** | 평지 + 숲 + 강 | 필드에서 전투 |
| **공성전** | 성벽 + 성문 + 마을 | 거점 공격 |
| **해전** | 수면 + 섬 | 해안 지역 전투 |

---

## 턴 진행

### 행동 순서

```
속도 = 기본 5 + (병과 보너스) + (아이템 보너스)

빠른 순서부터 행동:
  유격(+3) > 책사(+1) > 장수(0) > 관료(0) > 예인(-1) > 장인(-2)
```

### 1턴에 가능한 행동 (택 1)

| 행동 | 설명 | 조건 |
|------|------|------|
| **이동 + 공격** | 이동 후 인접 적 공격 | 이동력 내 |
| **사격** | 원거리 공격 (이동 불가) | 기술 5+ |
| **계략** | 적에게 디버프 (이동 불가) | 지략 5+ |
| **도술** | 범위 공격 or 회복 (이동 불가) | 지략 8+ |
| **고무** | 인접 아군 사기 회복 (이동 가능) | 예인 병과 |
| **방어** | 이번 턴 방어력 2배 | 누구나 |
| **후퇴** | 전장 이탈 | 누구나 |

## 대미지 계산

### 근접 공격

```
대미지 = (공격자 무력 × 병과 배율) × (1 - 방어자 방어율)
  × 사기 보정 × 지형 보정 × 랜덤(0.8~1.2)

방어율 = 방어자 무력 × 0.05 + 지형 보너스 + 장비 보너스
  (최대 70%)
```

### 원거리 공격 (사격)

```
대미지 = (공격자 기술 × 병과 배율) × (1 - 방어자 방어율 × 0.5)
  × 거리 감쇠(1칸: 100%, 2칸: 80%, 3칸: 60%)

사거리 = 2 + (기술 >= 8 ? 1 : 0)
```

### 계략

```
성공률 = 50% + (시전자 지략 - 대상 지략) × 10%
  (최소 10%, 최대 90%)

효과:
  - 혼란: 1턴 행동 불가
  - 이간: 사기 -20
  - 화공: 주변 1칸 대미지 (숲 지형 2배)
  - 매복: 다음 공격 대미지 2배
```

### 도술 (지략 8+)

```
성공률 = 100% (지략 8+ 보장)

효과 (택 1):
  - 뇌격: 단일 대상 고대미지 (지략 × 3)
  - 치유: 아군 1명 HP 30% 회복
  - 결계: 아군 1명 1턴 피해 무효
```

## 사기 시스템

수호지의 핵심 시스템. 사기가 0이 되면 괴멸.

| 사기 | 상태 | 효과 |
|------|------|------|
| 80-100 | 고양 | 공격력 +10% |
| 50-79 | 보통 | 기본 |
| 20-49 | 저하 | 공격력 -20%, 계략 저항 -10% |
| 1-19 | 궤멸 직전 | 공격력 -40%, 도주 가능성 |
| 0 | 괴멸 | 전투 이탈 (포로 or 도주) |

### 사기 변동

| 이벤트 | 사기 변동 |
|--------|----------|
| 적 처치 | +10 |
| 계략 적중 | +5 |
| 아군 피격 | -5 |
| 아군 사망 | -15 |
| 주군 피격 | -20 |
| 예인 고무 | +10~20 (문화 스탯 비례) |
| 턴 경과 (공격 측) | -1/턴 |

## 일기토

무력 7+ 장수끼리 인접 시 발동 가능. 양측 동의 시 1:1 전투.

```
일기토 라운드:
  1. 양측 행동 선택: 공격 / 수비 / 필살
  2. 판정:
    - 공격 vs 수비: 공격자 (무력 × 0.8) 대미지
    - 공격 vs 공격: 양측 (무력 × 1.0) 대미지
    - 필살 vs 수비: 필살자 (무력 × 2.0) 대미지
    - 필살 vs 공격: 공격자 (무력 × 1.5) 대미지 (필살 빗나감)
  3. HP 0 시 패배 → 포로 or 사망
```

## 전투 결과

| 결과 | 조건 | 보상 |
|------|------|------|
| **완승** | 적 전원 괴멸/포로 | 거점 점령 + 포로 + 물자 100% |
| **승리** | 적 주력 괴멸 | 거점 점령 + 물자 60% |
| **무승부** | 30턴 소진 | 양측 후퇴 |
| **패배** | 아군 주력 괴멸 | 후퇴, 물자 손실 |
| **대패** | 아군 전원 괴멸 | 거점 상실 위험, 주군 포로 위험 |

### 포로 처리

포로가 된 적 장수에 대해:

| 선택 | 효과 |
|------|------|
| **등용** | 영입 시도 (덕망 기반 확률) |
| **석방** | 명성 +5, 해당 인물 호감도 상승 |
| **감금** | 매 턴 등용 재시도 가능, 명성 -2 |
