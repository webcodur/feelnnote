# 05. 아이템 시스템

## 핵심 원칙

아이템은 DB의 `contents` 테이블(도서, 영화, 게임, 음악)에서 가져온다.
셀럽이 실제로 언급/추천한 콘텐츠가 게임 내 장비/소비 아이템이 된다.

---

## 아이템 분류

### 콘텐츠 타입 → 아이템 카테고리

| contents.type | 아이템 카테고리 | 장착 슬롯 | 효과 계열 |
|--------------|----------------|----------|----------|
| **BOOK** | 서적 (書) | 서적 슬롯 | 지략/덕망/문화 상승 |
| **VIDEO** | 그림두루마리 (畵) | 보물 슬롯 | 무력/지략/사기 상승 |
| **GAME** | 병법서 (兵) | 서적 슬롯 | 무력/기술/전투 특수 |
| **MUSIC** | 악보 (樂) | 보물 슬롯 | 문화/덕망/사기 회복 |

---

## 장착 시스템

캐릭터당 장착 슬롯:

```
[무기 슬롯]   - 병과별 고정 무기 (아이템 아님, 병과에 의해 결정)
[서적 슬롯]   - BOOK 또는 GAME 아이템 1개
[보물 슬롯]   - VIDEO 또는 MUSIC 아이템 1개
```

---

## 아이템 등급 산정

아이템의 등급은 **원작의 영향력**으로 결정한다.
셀럽의 total_score 평균이 높을수록 강한 아이템.

```
아이템 점수 = 해당 콘텐츠를 언급한 셀럽들의 total_score 평균
```

| 등급 | 점수 | 색상 | 예시 |
|------|------|------|------|
| **전설** | 65+ | 금색 | 일리아스 (알렉산더+여러 학자가 언급) |
| **영웅** | 50-64 | 보라 | 군주론, 손자병법 |
| **희귀** | 35-49 | 파랑 | 파우스트, 리어왕 |
| **일반** | 20-34 | 초록 | 대부분의 도서/영화 |
| **평범** | 0-19 | 흰색 | 1명만 언급한 콘텐츠 |

---

## 스탯 보너스 계산

### BOOK (서적)

```
지략 보너스 = floor(아이템 점수 / 20)    // 0~3
덕망 보너스 = floor(아이템 점수 / 25)    // 0~2
문화 보너스 = floor(아이템 점수 / 30)    // 0~2
```

### VIDEO (그림두루마리)

```
무력 보너스 = floor(아이템 점수 / 25)    // 0~2
지략 보너스 = floor(아이템 점수 / 25)    // 0~2
전투 사기 보너스 = floor(아이템 점수 / 15)  // 0~4
```

### GAME (병법서)

```
무력 보너스 = floor(아이템 점수 / 20)    // 0~3
기술 보너스 = floor(아이템 점수 / 20)    // 0~3
특수: 공성전 대미지 +10%
```

### MUSIC (악보)

```
문화 보너스 = floor(아이템 점수 / 20)    // 0~3
덕망 보너스 = floor(아이템 점수 / 30)    // 0~2
특수: 매 전투 턴 사기 +2 (고무 효과)
```

---

## 아이템 획득

### 1. 초기 보유

- 게임 시작 시 `user_contents`에서 주군의 콘텐츠 목록을 가져온다
- 주군이 언급한 콘텐츠 중 랜덤 1~3개를 초기 아이템으로 지급

### 2. 영입 보너스

- 새 인재 영입 시, 해당 인재의 콘텐츠 중 랜덤 1개를 아이템으로 획득
- "이 인물이 추천한 작품을 입수했습니다"

### 3. 도서관/학당 연구

- 도서관 건물 보유 시, 매 턴 일정 확률로 새 아이템 발견
- 발견 확률: `10% + (담당관 지략 × 2%)`

### 4. 전투 노획

- 전투 승리 시 적 아이템 획득 가능 (30%)
- 거점 점령 시 해당 거점 보유 아이템 전량 획득

### 5. 교역

- 교역소 보유 시, 금화로 아이템 구매 가능
- 매 턴 교역 목록 갱신 (랜덤 3~5개)

---

## 특수 아이템 (비콘텐츠)

콘텐츠 외에 게임 내 생성되는 특수 아이템:

| 아이템 | 효과 | 획득 |
|--------|------|------|
| **명마** | 이동력 +1 | 이벤트/교역 |
| **명검** | 무력 +2 | 무기고 제작/이벤트 |
| **부적** | 도술 1회 사용 (소모) | 사원 제작 |
| **군량** | 출정 식량 소모 -30% | 농장 잉여 |
| **보물** | 영입 시 선물 (매력 +20%) | 전투 노획/이벤트 |
