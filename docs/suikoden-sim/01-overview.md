# 01. 게임 개요

## 컨셉

**장르**: 턴제 전략 시뮬레이션 (웹게임)
**테마**: 역사 속 실존 인물들이 세력을 형성하여 문명을 통일하는 가상 역사
**참조**: 코에이 수호전 천도 108성 (1996)

---

## 핵심 게임 루프

```
[시작] 주군 선택
  → 거점 확보 (초기 지역 배정)
  → 인재 영입 (탐색 → 등용)
  → 거점 경영 (건설, 자원 관리, 내정)
  → 전투/외교 (인접 세력과 대결)
  → 영토 확장
  → [승리] 전 지역 통일
```

---

## 승리 조건

| 조건 | 설명 |
|------|------|
| **통일 승리** | 모든 지역을 지배하에 둔다 |
| **문화 승리** | 문화 영향력이 전체 세력 합산의 60% 이상 도달 |
| **외교 승리** | 전체 세력의 2/3 이상과 동맹 체결 |

---

## 패배 조건

| 조건 | 설명 |
|------|------|
| **거점 상실** | 자신의 모든 거점이 함락 |
| **턴 초과** | 제한 턴(100턴) 내 미통일 시 "역사의 망각" 엔딩 |

---

## 게임 흐름 (1턴 구조)

```
1턴 = 1계절 (봄/여름/가을/겨울)

[내정 페이즈]
  - 건설 / 철거
  - 자원 배분 (식량, 금, 지식)
  - 인재 탐색 / 등용

[행동 페이즈]
  - 이동 / 주둔
  - 전투 개시
  - 외교 (동맹, 정전, 항복 권유)

[이벤트 페이즈]
  - 랜덤 이벤트 (자연재해, 발명, 반란 등)
  - 아이템 발견
  - 특수 이벤트 (캐릭터 고유)

[정산 페이즈]
  - 자원 수입/지출
  - 사기 변동
  - AI 세력 행동
```
