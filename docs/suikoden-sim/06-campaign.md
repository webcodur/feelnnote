# 06. 캠페인 (맵과 세력)

## 세계관

"시간의 경계가 무너진 세계."
고대부터 근대까지의 위인들이 하나의 세계에 공존한다.
플레이어는 한 명의 주군을 선택하여 이 혼돈의 세계를 통일한다.

---

## 맵 구성

### 지역 (7개)

| 지역 | 거점 수 | 지형 특성 | 주요 국적 |
|------|---------|----------|----------|
| **동아시아** | 3 | 평지+강+산 | CN, KR, JP, MN |
| **남아시아** | 2 | 평지+정글+강 | IN |
| **중동** | 2 | 사막+오아시스+산 | SA, TR, EG, IL |
| **지중해** | 3 | 해안+섬+평지 | IT, GR |
| **서유럽** | 3 | 평지+숲+강 | FR, GB, DE, NL |
| **북유럽/동유럽** | 2 | 숲+설원+강 | SE, RU, PL, AT |
| **신대륙** | 2 | 평지+숲+해안 | US |

### 거점 간 연결

```
동아시아 ── 남아시아 ── 중동 ── 지중해 ── 서유럽
    │                      │                │
    └──────────────────────┘          북유럽/동유럽
                                          │
                              서유럽 ── 신대륙
```

- 인접 지역 간에만 이동/공격 가능
- 해로(海路): 동아시아 ↔ 중동, 지중해 ↔ 신대륙 (해군 필요)

## 세력 생성 (게임 시작 시)

### 플레이어 세력

1. 캐릭터 풀에서 주군 1명 선택
2. 주군의 `nationality` 기반으로 초기 지역 자동 배정
3. 같은 지역에서 랜덤 초기 인재 2~4명 배정

### AI 세력 (자동 생성)

```
AI 세력 수 = 난이도별 (3 / 5 / 7)

생성 규칙:
1. 각 지역별 가장 total_score가 높은 인물을 AI 주군 후보로
2. 플레이어 지역과 겹치지 않는 지역에 배치
3. 주군 국적과 지역이 일치하도록 우선 배정
4. 각 AI에 2~4명의 초기 인재 배정 (같은 지역 인물 우선)
```

### 방랑 인재

- 세력에 배정되지 않은 나머지 인물들
- 국적 기반으로 해당 지역에 방랑 상태로 배치
- 매 턴 일정 확률로 인접 지역으로 이동

## 외교

### 외교 행동 (관료 병과 또는 외교관저 필요)

| 행동 | 비용 | 효과 |
|------|------|------|
| **동맹 제안** | 금 200 | 성공 시 상호 불가침 + 공동 작전 가능 |
| **정전 협정** | 금 100 | 5턴간 상호 불가침 |
| **항복 권유** | - | 상대 전력 30% 이하 시 가능 |
| **조공** | 금/식량 | 관계도 상승 |
| **이간** | 금 300 | 두 AI 세력 간 관계 악화 |

### 외교 성공률

```
성공률 = 기본 30%
  + (사절 지략 × 3%)
  + (주군 덕망 × 2%)
  + (전력 차이 보정)
  + (기존 관계도 보정)
```

## AI 행동 패턴

AI 주군의 DB 스탯에 따라 행동 성향이 결정된다.

| 주군 스탯 최고치 | AI 성향 | 행동 |
|----------------|---------|------|
| 무력 | **정복형** | 적극 공격, 영토 확장 우선 |
| 지략 | **책략형** | 계략·외교 우선, 이간·동맹 다용 |
| 경영 | **경제형** | 내정 우선, 자원 축적 후 후반 공격 |
| 덕망 | **인덕형** | 영입 우선, 대군 편성 후 물량전 |
| 문화 | **문화형** | 문화 승리 추구, 방어적 |

## 이벤트

### 계절 이벤트

| 계절 | 이벤트 |
|------|--------|
| 봄 | 풍년: 식량 +50% / 역병: 병력 -10% |
| 여름 | 홍수: 강 인접 거점 피해 / 풍작: 식량 +30% |
| 가을 | 수확제: 민심 +10 / 유민 유입: 인구 증가 |
| 겨울 | 한파: 식량 소모 2배 / 설원 이동 불가 |

### 특수 이벤트 (캐릭터 기반)

```
조건: 특정 캐릭터 조합이 같은 세력에 있을 때

예시:
- 플라톤 + 아리스토텔레스 → "아카데메이아" 이벤트
  → 학당 효율 2배 (5턴)
- 나폴레옹 + 웰링턴 → "숙명의 대결" 이벤트
  → 양측 전투 시 무력 +2
- 이순신 + 세종대왕 → "조선의 빛" 이벤트
  → 한국 국적 인재 전원 사기 +20
```

이벤트 조합은 하드코딩하지 않고, 추후 DB 테이블로 관리 가능.
