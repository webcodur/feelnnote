# Feel&Note

## 당신의 문화생활이 하나의 서사가 되는 곳

---

## 문제

우리는 매일 책을 읽고, 영화를 보고, 게임을 한다.

그런데 그 경험은 어디로 가는가?

왓챠피디아에 별점을 남기고, 인스타에 감상 한 줄 올리고, 노션에 독서 기록을 정리한다. 흩어진 기록들. 연결되지 않는 경험들. 결국 우리가 무엇을 느꼈는지, 어떻게 변했는지는 시간 속에 사라진다.

**기존 서비스들의 한계:**

| 서비스 | 한계 |
|--------|------|
| 왓챠피디아 | 영화 중심, 단순 평점 |
| 리디북스 | 도서만, 플랫폼 종속 |
| 굿즈노트 | 개인 기록, 공유 불가 |
| 노션/옵시디언 | 직접 구축 필요, 진입장벽 |

**시장은 통합 솔루션을 원한다.**

---

## 해결책

**Feel&Note는 모든 문화 콘텐츠를 한 곳에서 기록하고, 깊이 있게 분석하며, 취향이 맞는 사람들과 연결되는 플랫폼이다.**

### 핵심 가치

```
📚 통합        하나의 앱에서 도서, 영화, 드라마, 애니, 게임, 공연까지
📝 깊이        별점을 넘어 리뷰, 노트, 창작까지
🎮 재미        티어리스트, 블라인드 게임으로 즐기는 기록
🏆 동기부여    점수와 칭호로 지속하는 기록 습관
👥 연결        취향이 맞는 사람들과의 자연스러운 만남
```

---

## 제품

### 1. 기록관 (Archive)

**"내 문화생활의 아카이브"**

모든 콘텐츠를 한 곳에서 관리한다. 검색으로 추가하고, 상태와 진행도를 기록한다.

- **6개 카테고리**: 도서, 영화, 드라마, 애니메이션, 게임, 공연
- **상태 관리**: 관심 → 감상 중 → 완료
- **진행도 추적**: 10% 단위 진행률, 자동 상태 전환
- **다양한 뷰**: 그리드/리스트, 카테고리별 필터
- **플레이리스트**: 나만의 컬렉션 큐레이션

### 2. 기록 시스템

**"경험을 기록으로 남기다"**

#### 리뷰
별점과 감상평. 완료 후 남기는 종합 평가.

#### 노트
과정 속 메모. 읽으면서, 보면서 떠오르는 생각들.
- 섹션 기반 에디터
- 진행도와 연동
- 태그 분류

#### 창작
상상의 확장. 콘텐츠를 넘어 나만의 이야기.
- What If: 다른 선택, 다른 결말
- 매체 전환: "이 소설이 영화가 된다면"
- OST 상상: 장면별 음악 선곡

### 3. 놀이터 (Playground)

**"기록을 놀이로"**

#### 티어리스트
S/A/B/C/D 등급으로 내 작품들을 랭킹.
- 드래그 앤 드롭 편집
- 카테고리별/커스텀 주제
- 이미지 저장, SNS 공유
- 트렌딩/최신 피드

#### 블라인드 게임
감상평만 보고 작품 맞추기.
- 내 기록 기반 퀴즈 자동 생성
- 힌트 시스템
- 스트릭 보너스
- 리더보드

### 4. 업적 시스템

**"기록이 성장으로"**

활동에 점수가 쌓이고, 조건 달성 시 칭호가 해금된다.

#### 점수
| 활동 | 점수 |
|------|------|
| 콘텐츠 추가 | +1 |
| 리뷰 작성 | +3 |
| 노트 작성 | +2 |
| 창작 작성 | +5 |

#### 칭호 카테고리
| 카테고리 | 예시 |
|----------|------|
| 기록량 | "백 권의 무게", "천 편의 기억" |
| 다양성 | "르네상스인", "장르 유목민" |
| 꾸준함 | "30일 연속", "1년 개근" |
| 깊이 | "논문급 리뷰어", "디테일 장인" |
| 특수 | "첫 발자국", "얼리어답터" |

### 5. 검색

**"원하는 것을 빠르게"**

- **콘텐츠 검색**: 외부 API 연동, 카테고리별
- **내 기록 검색**: 기록관 내 검색
- **사용자 검색**: 취향이 맞는 사람 찾기
- **태그 검색**: 관심사 기반 탐색

## 비전

**Feel&Note는 단순한 리뷰 앱이 아니다.**

문화생활 전체를 기록하고,
과정 속 감동을 보존하며,
상상을 펼치고,
취향이 맞는 사람들과 깊이 있게 연결되는 공간.

빌 게이츠의 Gates Notes처럼, 한 사람의 문화생활 전체가 하나의 서사가 되는 곳.

**그것이 Feel&Note다.**
