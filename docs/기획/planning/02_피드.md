# 피드 & 소셜

## 개요

팔로우한 사람들의 문화생활이 흐르는 타임라인. 취향이 맞는 사람들과 연결되는 공간.

> **현재 상태**: UI 구현 완료, 소셜 API 개발 예정

---

## 소셜 시스템

### 관계 유형

| 관계 | 설명 |
|------|------|
| 팔로우 | 일방향 관계, 피드에 노출 |
| 팔로워 | 나를 팔로우하는 사람 |
| 친구 | 상호 팔로우 (맞팔) |

### 기능

| 기능 | 설명 | 상태 |
|------|------|------|
| 팔로우 | 유저 팔로우/언팔로우 | 예정 |
| 차단 | 특정 유저 차단 | 예정 |
| 프로필 | 다른 유저 프로필 조회 | 예정 |

---

## 피드

### 피드 유형

| 유형 | 설명 |
|------|------|
| 메인 피드 | 팔로우 + 추천 혼합 |
| 팔로잉 피드 | 팔로우한 유저만 |

### 피드 콘텐츠

| 유형 | 노출 조건 |
|------|-----------|
| 리뷰 | 전체 공개 또는 팔로워 공개 |
| 노트 | 전체 공개 또는 팔로워 공개 |
| 창작 | 전체 공개 |
| 티어리스트 | 공개 설정 |

### 정렬

| 방식 | 설명 |
|------|------|
| 스마트 | 관계 + 시간 가중치 |
| 최신순 | 순수 시간순 |

### 필터

- 활동 유형 (전체, 리뷰, 노트, 창작)
- 카테고리 (전체, 도서, 영화...)
- 기간 (전체, 오늘, 이번 주, 이번 달)

---

## 상호작용

### 좋아요

- 기록에 좋아요 토글
- 좋아요 수 표시

### 댓글

- 1depth 댓글
- 댓글 작성/삭제

### 공유

- 링크 복사
- SNS 공유

## 데이터 모델 (예정)

### follows

| 필드 | 타입 | 설명 |
|------|------|------|
| follower_id | UUID | 팔로우하는 사람 |
| following_id | UUID | 팔로우당하는 사람 |
| created_at | TIMESTAMPTZ | 팔로우 시점 |

### blocks

| 필드 | 타입 | 설명 |
|------|------|------|
| blocker_id | UUID | 차단하는 사람 |
| blocked_id | UUID | 차단당하는 사람 |
| created_at | TIMESTAMPTZ | 차단 시점 |

### record_likes

| 필드 | 타입 | 설명 |
|------|------|------|
| user_id | UUID | 좋아요한 사람 |
| record_id | UUID | 기록 ID |
| created_at | TIMESTAMPTZ | 좋아요 시점 |

### record_comments

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| user_id | UUID | 작성자 |
| record_id | UUID | 기록 ID |
| content | TEXT | 댓글 내용 |
| created_at | TIMESTAMPTZ | 작성일 |
