# 놀이터 (Playground)

## 개요

기록을 놀이로 확장하는 공간. 티어리스트로 내 취향을 정리하고, 블라인드 게임으로 기록을 즐긴다.

---

## 티어리스트

### 개요

S/A/B/C/D 5단계 등급으로 내가 경험한 작품들을 랭킹하는 기능.

### 등급 체계

| 등급 | 의미 | 색상 |
|------|------|------|
| S | 최고, 인생작 | #ffd700 (금색) |
| A | 매우 좋음 | #b57cff (보라) |
| B | 좋음 | #4d9fff (파랑) |
| C | 보통 | #50c878 (초록) |
| D | 별로 | #808080 (회색) |

### 기능

| 기능 | 설명 |
|------|------|
| 생성 | 이름, 카테고리로 새 티어리스트 생성 |
| 편집 | 드래그 앤 드롭으로 등급 배치 |
| 공개 | 공개/비공개 설정 |
| 공유 | 이미지 저장, SNS 공유, 링크 공유 |

### 리스트 종류

| 유형 | 설명 |
|------|------|
| 전체 | 모든 카테고리 통합 |
| 카테고리별 | 도서/영화/드라마 등 개별 |
| 커스텀 | 사용자 정의 주제 |

### 피드

| 탭 | 설명 |
|-----|------|
| 트렌딩 | 인기 티어리스트 |
| 최신 | 최근 공개된 티어리스트 |
| 팔로잉 | 팔로우한 유저의 티어리스트 |
| 내 티어리스트 | 내가 만든 목록 |

### UI

```
┌─────────────────────────────────────────┐
│ 🧩 놀이터                               │
│                                         │
│ [티어리스트] [블라인드 게임]  [+ 새 리스트]│
├─────────────────────────────────────────┤
│ [트렌딩] [최신] [팔로잉] [내 티어리스트]  │
├─────────────────────────────────────────┤
│ ┌─────────────┐ ┌─────────────┐         │
│ │ S │ ■■■     │ │ S │ ■■      │         │
│ │ A │ ■■■■    │ │ A │ ■■■■    │         │
│ │ B │ ■■      │ │ B │ ■■■     │         │
│ │ 리스트 제목 │ │ 리스트 제목 │         │
│ │ @유저 · 도서│ │ @유저 · 영화│         │
│ │ ♡ 24  ⑂ 3  │ │ ♡ 18  ⑂ 1  │         │

└─────────────────────────────────────────┘
```

## 블라인드 게임

### 개요

제목/썸네일 없이 감상평만 보고 작품을 맞추는 퀴즈 게임.

### 게임 방식

1. 내 기록(리뷰/인용문)에서 랜덤 선택
2. 제목/표지 없이 내용만 표시
3. 작품명 추측
4. 정답/오답 판정

### 문제 소스

| 소스 | 설명 |
|------|------|
| 내 기록 | 내가 작성한 리뷰/인용문 |
| 팔로잉 | 팔로우한 유저의 공개 기록 |
| 전체 | 전체 공개된 기록 |

### 난이도

| 난이도 | 별점 | 설명 |
|--------|------|------|
| 쉬움 | ★☆☆☆☆ | 힌트 많음, 명확한 내용 |
| 보통 | ★★★☆☆ | 적당한 난이도 |
| 어려움 | ★★★★★ | 힌트 적음, 추상적 내용 |

## 데이터 모델

### tier_lists

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| user_id | UUID | 생성자 |
| name | TEXT | 리스트 이름 |
| category | TEXT | 카테고리 (nullable) |
| tiers | JSONB | 등급별 콘텐츠 ID 배열 |
| is_public | BOOLEAN | 공개 여부 |
| created_at | TIMESTAMPTZ | 생성일 |
| updated_at | TIMESTAMPTZ | 수정일 |

### blind_game_scores

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| user_id | UUID | 플레이어 |
| score | INTEGER | 점수 |
| streak | INTEGER | 연속 정답 |
| played_at | TIMESTAMPTZ | 플레이 시간 |
