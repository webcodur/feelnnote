# 리뷰 플랫폼 프리셋 정리 (2026-02-09 기준)

리뷰 플랫폼에서 유저들이 쉽게 리뷰를 작성할 수 있도록 **공통 프리셋**과 **카테고리별 특화 프리셋**을 제공합니다.  
특히 **자격증** 카테고리는 **자격증 그 자체**(시험 난이도, 공부 과정, 합격 가치, 실무 활용도 등)에 초점을 맞췄습니다.

## 1. 공통 프리셋  
모든 카테고리(도서 · 음악 · 영상 · 게임 · 자격증)에 공통으로 적용 가능한 기본 프리셋입니다.  
총 12개로, 유저가 가장 자주 사용할 만한 감정·평가 중심으로 구성했습니다.

- [유익해요]  
- [감동적이에요]  
- [잘 만들어짐]  
- [다시 볼/할 만해요]

- [생각할 점 많아요]

## 3. 추천 구현 방식 (요약)

1. **기본 화면**  
   → **공통 프리셋** 10~12개만 먼저 노출 (가장 범용적인 것들 위주)

2. **카테고리 선택 시**  
   → 공통 프리셋 + 해당 카테고리 **특화 프리셋** 5~8개 추가 노출

3. **프리셋 선택 UX**  
   - 여러 개 선택 가능 (태그처럼 누적)  
   - 선택 후 **AI 자동 문장 생성** 예시  
     - [강력 추천] + [재미있어요] + [중독성 강함]  
       → “이 게임 강력 추천해요! 진짜 재미있고 중독성 쩔어서 시간 가는 줄 몰랐습니다.”  
     - [합격 가치 높음] + [실무 바로 써먹음] + [난이도 적당]  
       → “합격 가치 진짜 높아요! 실무에 바로 써먹을 수 있고 난이도도 적당해서 강추합니다.”

4. **추가 고려사항**  
   - 각 프리셋에 **이모지** 붙이기 (예: [강력 추천] 👍👍, [어려워요] 😓)  
   - 프리셋 선택 후 **“왜 그렇게 느꼈나요?”** 후속 프롬프트로 자세한 리뷰 유도  
   - 운영 후 **유저 데이터** 기반으로 인기 프리셋 순위·추가 업데이트

필요 시 카테고리별 프리셋을 더 늘리거나, 특정 자격증군(IT·공무원·외국어 등)별로 세분화할 수도 있습니다.
